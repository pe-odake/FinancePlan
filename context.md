# Finance Plan

É um site que a pessoa pode se organizar financeiramente onde ele pode cadastrar suas receitas e despesas, e também pode simular investimentos diversos, como na bolsa de valores e em bancos.

# Organização do Projeto

frontend - src - components (componentes reutilizáveis) - pages (crie todas as telas aqui) - services (serviços que serão usados no projeto) - utils (funções utilitárias) - App.tsx - index.tsx - public - package.json - vite.config.ts
backend - src - core - models (crie aqui os modelos do banco de dados) - routers (crie aqui os routers) - schemas (crie aqui os schemas) - service (crie aqui os serviços) - main.py - requirements.txt
